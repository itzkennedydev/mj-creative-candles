# Stitch Please

A modern e-commerce platform for custom embroidery and personalized designs, located in Moline, IL.

## Overview

Stitch Please offers high-quality embroidery services with a focus on:
- Custom embroidery and personalization
- Fast turnaround times without compromising quality
- Wide range of products including tops, bottoms, hats, and accessories
- Professional services for brands and businesses

## Features

- **Responsive Design**: Fully optimized for all devices and screen sizes
- **Modern UI**: Built with Next.js 13, Tailwind CSS, and Shadcn UI components
- **Performance Focused**: Optimized loading times and smooth animations
- **Category Navigation**: Easy browsing through different product categories
- **Custom Design Tools**: Create your own personalized designs
- **Brand Showcase**: Featured work with notable brands and organizations

## Tech Stack

- Next.js 13 (App Router)
- TypeScript
- Tailwind CSS
- Shadcn UI
- Radix UI Primitives
